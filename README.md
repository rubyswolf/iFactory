# iFactory

iFactory by Dart Spark is an Electron desktop for Windows for building audio plugins with
the Codex CLI and iPlug2 plugin framework.

## CLI
iFactory comes with a custom ifact CLI tool designed for Codex to use which links directly to the running app to automate functions and print documentation.
It must be activated with `npm link`.

## Run

```powershell
npm install
npm start
```
